== Description ==
 
with this plugin users can login with his/her pod's account.
 
== Installation ==

1. Upload whole project to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. open the podsso setting and configure the plugins setting
